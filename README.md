## Vite + Vue 3 + TypeScript + Element Plus + Pinia

### 预览地址

[perlou.github.io/vue3-ts-tpl](https://perlou.github.io/vue3-ts-tpl)

### 技术栈

- ⚡ [Vite 6](https://vite.dev/) - 下一代前端构建工具
- 🖖 [Vue 3.5](https://vuejs.org/) - 渐进式 JavaScript 框架
- 🔤 [TypeScript 5.7](https://www.typescriptlang.org/) - 类型安全
- 🎨 [Element Plus 2.9](https://element-plus.org/) - Vue 3 组件库（按需自动导入）
- 🍍 [Pinia](https://pinia.vuejs.org/) - 状态管理
- 🌐 [Axios](https://axios-http.com/) - HTTP 请求
- 📅 [Day.js](https://day.js.org/) - 日期处理
- 🧪 [Vitest](https://vitest.dev/) - 单元测试
- 📦 [pnpm](https://pnpm.io/) - 包管理器

### 快速开始

```bash
pnpm install
pnpm dev
```

### 项目打包

```bash
pnpm build
```

### 单元测试

```bash
pnpm test
```

### 代码检查

```bash
pnpm lint
pnpm lint:fix
```
